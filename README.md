# X-POCKET

> [Next-js-Boilerplate](https://github.com/ixartz/Next-js-Boilerplate)

## Pocket Universe

### Requirements

- Node.js 22+ and npm

### Vercel

```shell
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 链接项目
vercel link

# 列出环境变量
vercel env ls

# 下载环境变量
vercel env pull .env.local

```

### Getting started

```shell
npm install

npm run dev

npm run build

npm run start

npm run commit
```
